# Prismo Brand Guidelines

Official brand assets for Prismo by [diShine Digital Agency](https://dishine.it).

📌 **Quick links**: [README](../../README.md) · [User Guide](../../docs/GUIDE.md) · [Export Guide](../../toolkit/export/README.md)

## Brand Colors

| Name | Hex | Usage |
|------|-----|-------|
| Prismo Purple | `#6C5CE7` | Primary brand color, headers, buttons |
| Teal Accent | `#00CEC9` | Secondary highlights, links |
| Charcoal | `#2D3436` | Body text, dark backgrounds |
| Snow | `#F5F6FA` | Light backgrounds, cards |
| Success Green | `#00B894` | Passed checks, low severity |
| Warning Amber | `#FDCB6E` | Medium severity |
| Danger Coral | `#E17055` | High severity |
| Critical Red | `#D63031` | Critical severity |
| Info Blue | `#74B9FF` | Informational badges |

## Severity Classification

| Level | Color | Icon | Usage |
|-------|-------|------|-------|
| Critical | `#D63031` | 🔴 | Immediate action required |
| High | `#E17055` | 🟠 | Fix soon |
| Medium | `#FDCB6E` | 🟡 | Plan to address |
| Low | `#00B894` | 🟢 | Minor improvement |
| Info | `#74B9FF` | 🔵 | Informational only |

## Typography

- **Headings**: Inter, Segoe UI, sans-serif (Bold)
- **Body**: Inter, Segoe UI, sans-serif (Regular)
- **Code**: JetBrains Mono, monospace

## Logo

Logo files should be placed here as:
- `prismo-logo-full.svg` — Full logo with wordmark
- `prismo-icon.svg` — Icon only (for favicons)
